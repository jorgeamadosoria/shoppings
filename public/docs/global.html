    zipper(file) → {function}

    This function creates a lambda that zips a file containing a JSON representation of all the entites of the application. Useful as backup
    Parameters:
    Name Type Description
    file string the name of the JSON file, corresponding to the entity to be zipped
    Returns:
    function - a lambda corresponding to a zipping action to respond to a fulfilled promise
    Example
    zipper("brands.json")
