    inner userCallback(token, refreshToken, profile, done) → {User}
    config/passport.js, line 49

    This function authenticates an users by their Google OAuth credentials. If the user logs in for the first time, a record is created on the database for them. This allows later change of role.
    Parameters:
    Name Type Description
    token string google OAuth token
    refreshToken string google OAuth refresh token
    profile Object google profile scope object
    done function authentication callback to verify users
    Returns:
    User - the authenticated user or an error message if authentication fails
