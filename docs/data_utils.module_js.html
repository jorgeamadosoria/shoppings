    static createCrudRouter(service, entity, readRoles, writeRoles, formMw) → {Object}
    data/utils.js, line 27

    This function creates a CRUD Router, that is, a Router to manage entities other than the main one.
    Parameters:
    Name Type Description
    service string The corresponding service to the entity we want to CRUD
    entity string The corresponding service to the entity we want to CRUD
    readRoles Array.<string> Array of roles for read endpoints (list, detail)
    writeRoles Array.<string> Array of roles for write endpoints (insert, update, delete)
    formMw Object Middleware for the "form" url. This is different for each router because each form requires different nomenclator entities
    Returns:
    Object - a fully configured CRUD router for the entity specified
    Example
    createCrudRouter(service, "brands", ["reviewer", "user", "admin"], ["user", "admin"], function(req, res, next) { ... });
